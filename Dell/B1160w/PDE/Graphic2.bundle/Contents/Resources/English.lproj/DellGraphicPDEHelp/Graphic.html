Graphics page

The Graphics page defines the graphic setting for the print job.


Resolution

Resolution defines the quality of the point output : the number of dots per inch.

Quality

Quality defines the quality of the print output in the simplest form.

Image Mode

Image Mode allows users to enhance printouts.

Color Mode

Color Mode allows the user to print an image colored or grayscale.

Black Optimization
Monochrome pages are printed with black toner only, if this option is selected.

Page Color Setting
You can set Color/Grayscale printing options for each page. This feature might not work correctly with combination of other additional features. Some applications might not support this feature.

Darkness

The Darkness option adjusts the toner density on print outputs.
